技术栈

前端

  • React 18 + TypeScript
  • Vite 5（HMR 热更新）
  • Three.js + STLLoader
  • 参数面板（滑块实时回写）

AI / Python

  • OpenAI SDK（兼容七牛 DeepSeek）
  • openai-agents SDK
  • System Prompt 强约束输出

后端

  • Node.js + Express
  • tsx watch 热重载
  • WebSocket（指数退避重连）
  • AbortController 并发控制

编译

  • OpenSCAD CLI（本地安装）
  • 临时文件隔离
  • queued/running/success/error 四态

核心功能

💬

自然语言建模

输入描述即可生成参数化 OpenSCAD 代码

🎚️

参数实时调节

滑块拖动 → 代码同步更新 → 自动重编译

🖼️

3D 实时预览

Three.js 加载 STL，支持旋转 / 缩放

🔁

会话持久化

WebSocket 同步，刷新不丢失历史对话

🛠️

错误恢复

编译失败显示结构化错误 + 一键 AI 修复

⚡

并发控制

连续调参只保留最后一次编译结果

不足与规划

当前不足

  • 编译依赖本地 OpenSCAD 安装
  • 3D 渲染缺 OrbitControls
  • 无用户认证体系
  • AI 修复未深度实现
  • 无 STL 导出按钮
  • 无自动化测试覆盖

后续规划

  • 集成 openscad-wasm，实现零依赖浏览器端编译

  • STL / SCAD 文件一键下载
  • AI 结构化错误定位修复
  • 端到端集成测试

快速启动

# 1. 克隆仓库
git clone https://github.com/dlfu121/ScadGenerator.git
cd ScadGenerator

# 2. 配置 API Key
cp .env.example .env
# 编辑 .env，填写 QINIU_DEEPSEEK_API_KEY

# 3. 安装依赖
npm run install:all

# 4. 启动
npm run dev

# 前端: http://localhost:5173
# 后端: http://localhost:5000
